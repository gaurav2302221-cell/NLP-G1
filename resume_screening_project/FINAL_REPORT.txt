HOW TO USE (UPDATED)
====================================================================

1. OPEN APPLICATION
   URL: http://localhost:8501
   (App is currently running in background)

2. INITIALIZE MODELS (First Time Only)
   - Click "Initialize Models" button in sidebar
   - Wait 10-15 seconds for BERT model to load
   - See confirmation message

3. UPLOAD RESUMES
   - Click file uploader
   - Select PDF files (single or multiple)
   - Click "Process Resumes"
   - See confirmation with file count

4. ENTER JOB DESCRIPTION
   - Paste complete job description
   - Include title, requirements, skills
   - Better to include detailed specifications

5. ANALYZE RESUMES
   - Click "Analyze Resumes" button
   - Wait 30-60 seconds (first run downloads BERT model)
   - Results appear automatically

6. REVIEW RESULTS
   - Check ranking table (sorted by score)
   - Review top candidate highlight
   - Check summary metrics
   - View visualizations
   - Examine skill analysis
   - Review job analysis

7. EXPORT RESULTS
   - Click "Download Results (Excel)" or "(CSV)"
   - File saves to your Downloads folder
   - Open in Excel/Sheets for further analysis

SUPPORT & TROUBLESHOOTING
====================================================================

Issue: App shows blank page
Solution: Refresh browser (Ctrl+R or Cmd+R)

Issue: Models not loading
Solution: Click "Initialize Models" in sidebar

Issue: Resumes won't process
Solution: Ensure PDFs are text-based (not scanned images)

Issue: No results showing
Solution: Check job description is not empty, has 10+ words

Issue: Slow performance
Solution: First run is slow (downloads BERT), subsequent runs faster

Issue: Export button missing
Solution: Complete analysis first, export section appears after
