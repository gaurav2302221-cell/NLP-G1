HOW TO USE THE WEB APPLICATION
==========================================================

1. OPEN IN BROWSER:
   - Go to: http://localhost:8501
   - Or click the link above

2. UPLOAD RESUMES:
   - Click "Browse files" in the Upload Resumes section
   - Select one or multiple PDF resume files
   - Click "Process Resumes" button
   - Status will show how many resumes were processed

3. ENTER JOB DESCRIPTION:
   - In the "Job Description" section
   - Paste the complete job description text
   - Include required skills and qualifications

4. ANALYZE:
   - Click the blue "Analyze Resumes" button
   - The system will:
     a) Process the job description
     b) Extract required skills
     c) Score each candidate
     d) Rank all candidates
     e) Display results with visualizations

5. VIEW RESULTS:
   - Candidate Rankings Table (ranked by score)
   - Top Candidate Highlight (best match)
   - Summary Metrics (average, min, max scores)
   - Visualizations (score charts)
   - Detailed Skill Analysis

6. EXPORT RESULTS:
   - Download as Excel (.xlsx)
   - Download as CSV (.csv)

==========================================================
TROUBLESHOOTING
==========================================================

Issue: App won't load
Solution: Check that StreamLit is running, wait 5-10 seconds for BERT model to load

Issue: PDFs not being parsed
Solution: Ensure PDFs are text-based (not scanned images)

Issue: "Skill not found" errors
Solution: Add new skills to SKILL_DICTIONARY in src/skill_extractor.py

Issue: Out of memory
Solution: Process fewer resumes at once or add more RAM
