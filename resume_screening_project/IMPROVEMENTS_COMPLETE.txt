HOW TO USE THE IMPROVED APPLICATION:

Step 1: OPEN APPLICATION
   URL: http://localhost:8501

Step 2: INITIALIZE MODELS (First Time Only)
   - Click "Initialize Models" button on sidebar
   - Wait 10-15 seconds for BERT model to load
   - See confirmation message when ready

Step 3: UPLOAD RESUMES
   - Click file uploader in "Upload Resumes" section
   - Select one or more PDF files
   - Click "Process Resumes" button
   - System processes and shows confirmation

Step 4: ENTER JOB DESCRIPTION
   - Paste job description in "Job Description" text area
   - Include title, requirements, skills
   - More detailed input = better results

Step 5: ANALYZE
   - Click blue "Analyze Resumes" button
   - Wait 30-60 seconds (first run loads BERT)
   - Progress spinner shows activity
   - Results appear automatically

Step 6: REVIEW RESULTS
   - Check professional ranking table
   - Read top candidate highlight
   - View summary metrics (Total, Average, Max, Min)
   - Review visualization charts
   - Examine detailed skill analysis

Step 7: EXPORT DATA
   - Click "Download Results (Excel)" or "(CSV)"
   - File saves with timestamp
   - Open in Excel or Google Sheets
